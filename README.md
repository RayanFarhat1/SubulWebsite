Project Description
Subul (سبل) is a responsive Arabic-language online bookstore built with React and modern CSS. The project delivers a seamless shopping experience with a culturally authentic design that blends Islamic art influences with modern UI/UX practices.
The platform allows users to browse books and other items, add them to a cart or favorites list, and complete purchases through a streamlined checkout flow. With a mobile-friendly navigation bar, dynamic cart functionality, and RTL (right-to-left) layout support, Subul is designed to feel intuitive and visually elegant for Arabic readers.

 Key Highlights
- Responsive navigation bar with integrated icons for account, favorites, and cart.
- Dynamic shopping cart with quantity adjustment, totals, and region-based shipping calculation.
- Favorites page with empty state handling.
- Checkout flow with order summary and user information form.
- Authentication pages (login/register) with RTL layout and Arabic typography.
